## Netflix Analytics Dashboard | Tableau Project
Interactive Tableau dashboard analyzing Netflix content trends, genres, ratings, and geographic distribution.

## Project Overview
This project analyzes Netflix content using Tableau to explore trends in movies, TV shows, genres, ratings, and countries.

## Tools Used
- Tableau Public
- Excel
- GitHub

## Key Features
- KPI Cards (Total Titles, Movies, TV Shows, Countries)
- Content Growth Over Time
- Top 10 Genres Analysis
- Ratings Distribution
- Country-Based Content Analysis
- Interactive Filters and Dashboard Actions

## Key Insights
- Movies make up the majority of Netflix content.
- Drama and Comedy are the most common genres.
- Netflix content increased significantly after 2015.
- USA contributes the largest share of content.
- TV-MA is the most common rating.

## Key Metrics
- Total Titles: ~8,800
- Movies: ~6,131
- TV Shows: ~2,676
- Countries: 100+
